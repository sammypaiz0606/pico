Introduction Pico.

A stupidly simple & blazing fast, flat

file CMS.

Making the web easy.


  • Stupidly Simple

    Pico makes creating and maintaining a website as simple as editing text files.


  • Blazing Fast

    Pico is seriously lightweight and doesn't use a database, making it super fast.


  • No Database

    Pico is a 'flat file' CMS, meaning no database woe's. no MySQL queries, nothing.

  • Markdown Formatting

    Edit your website in your favorite text editor using simplw Markdown formatting

  • Twig Templates

    Pico uses the Twig templating engine, for powerful and flexible themes

  • Open Source

    Pico is completely free and open source, released under the MIT license
